# Projetos na Linguagem C

Este repositório contém pequenos projetos em C criados para aprendizagem prática. Cada pasta apresenta um exemplo com foco em conceitos específicos da linguagem.

## Projetos incluídos

- **Algoritmos/bubblesort.c** – implementação de ordenação bubble sort em C.
- **Calculadora/** – calculadora de linha de comando que demonstra entrada/saída, operadores e estruturas de decisão. Consulte `Calculadora/README.md` para mais detalhes.

## Como usar

Navegue até o diretório do projeto que deseja compilar e siga as instruções de cada README.
